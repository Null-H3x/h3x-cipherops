# X25519 Elliptic-Curve Diffie-Hellman

## Mathematical Definition

Montgomery curve (Curve25519) scalar multiplication:

$$SS = \text{X25519}(a, B) = a \cdot B$$

Where $a$ is Alice's private key and $B$ is Bob's public key.

Dataset stores shared secret plus authenticated message envelope.
