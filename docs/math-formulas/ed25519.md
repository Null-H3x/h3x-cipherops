# Ed25519 Digital Signatures

## Mathematical Definition

Edwards-curve Digital Signature Algorithm on Curve25519:

$$\text{Signature} = (R, S) \text{ over message } m \text{ with private key } a$$

Verification recovers and validates using public key $A = aB$.

Dataset stores authenticated envelope `{message, signature}`.
